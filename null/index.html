What is HTML?

HTML,or a Hypertext markup language, is not a scripting language, it's a markup language!

Top 3 fun facts about HTML

  • 1. HTML was created by Tim Berners-Lee in 1991.
  • 2. HTML is the skeleton of every webpage you see on the internet.
  • 3. HTML5 introduced the (<)article(>) and (<)section(>) tags for better semantics.
